▼

Période : Novembre, Décembre 2024

Compétence(s) visée(s) : Programmation de switch, routeur, utilisation de serveur proxmox, utilisation de linux

Apprentissage(s) critiques : Configuration réseau avancée et sécurisation

Description du projet

Objectif : Concevoir et mettre en œuvre un réseau informatique sécurisé et structuré pour une PME (Fibre&Company) en pleine expansion. Principales tâches : Configurez des VLANs, du routage inter-VLAN, du DHCP, et RSTP sur des commutateurs Cisco. Mettre en place un serveur Debian (VM) avec des services réseau (SMB, DNS, TFTP, site web, utilisateurs). Renforcez la sécurité des informations sur le réseau (accès SSH, accès SSH, VLAN non utilisé, désactivation Telnet ou et /HTTP, etc.). Faire des tests de fonctionnement et de débit (Iperf3).

J’ai réalisé la partie programmation des switch 2960 et routeur dans le site web et aussi la réalisation d’un schéma logique de la maquette réseau.

Configuration switch/routeur ou
📋
Tests de performance

SAÉ 14 - Création de styles modernes pour une page web

▼

Période : Novembre 2024

Compétence(s) visée(s) : Développement de styles modernes et structurés pour le web

Apprentissages critiques : Utilisation de CSS avancé, responsive design et animations interactives

Description du projet

Ce projet consistait à concevoir des styles modernes et bien structurés pour une page web. Le travail incluait :

  • Réinitialisation des styles : Marges et espacements uniformes pour une base propre sur tous les navigateurs, évitant les styles par défaut.
  • Variables CSS : Utilisation de variables CSS pour gérer les couleurs, facilitant la maintenance et l’ajustement des styles globaux.
  • Responsive Design : Implémentation de media queries pour adapter l’affichage sur mobiles, tablettes et ordinateurs, garantissant une expérience utilisateur optimale.
  • Animations et interactivité : Création d’animations fluides (effets de survol, loader interactif) avec des transitions douces pour dynamiser la page.

Mon rôle et contributions personnelles

J’ai conçu les styles CSS, implémenté les variables et media queries, et ajouté des animations pour améliorer l’interactivité de la page.

💻 Code source disponible sur la page github

▼

Période : Décembre 2024

Compétence(s) visée(s) : Utilisation d’API web, programmation en Python et HTML

Apprentissages critiques : Intégration d’API RESTful, développement d’interfaces web dynamiques

Description du projet

Ce projet consistait à développer une application web utilisant l’API Pokémon pour afficher des informations sur les Pokémon. Les tâches incluaient :

  • Intégration de l’API : Utilisation de l’API Pokémon pour récupérer des données (noms, types, statistiques) via des requêtes HTTP en Python.
  • Développement web : Création d’une interface HTML/CSS pour afficher les données de manière claire et interactive.
  • Programmation backend : Utilisation de Python (avec Flask ou FastAPI) pour gérer les requêtes API et servir les données à l’interface.
  • Interactivité : Ajout de fonctionnalités comme la recherche de Pokémon et l’affichage dynamique des résultats.

Mon rôle et contributions personnelles

J’ai développé le backend Python pour interroger l’API Pokémon et conçu l’interface HTML pour afficher les résultats de manière dynamique.
